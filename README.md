# Rainy Days - E-commerce Concept

![image](https://github.com/aw3stm/html-css-ca-anna-vestman/blob/main/images/RainyDays_Home.png?raw=true)


## Description
This project was part of the HTML/CSS course at Noroff. Rainy Days is a webshop specializing in outdoor rainwear.

The primary goal of this project was to build a fully responsive website using only HTML and CSS. The user experience allows customers to navigate through the collection and click on specific jackets to access detailed product information and descriptions.

Key Objectives:

- Create a clean, responsive layout without the use of frameworks or JavaScript.

- Implement a product flow from landing page to individual product pages.

- Focus on semantic HTML and CSS architecture.

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone https://github.com/aw3stm/html-css-ca-anna-vestman.git
```

2. Install the dependencies:

```
npm install
```

### Running
To run the app, run the following commands:

```bash
npm run start
```
